Kruize - FizzBuzz

  • Navigate to the folder fizzbuzz\python\kruize
  • If you are getting an error where the line endings arent right for your system, run either dos2unix or unix2dos
  • Run the command {./run.sh #} and replace # with the max number you want it to count to.
  • The program will loop through all the numbers up to the number you gave and print something to the console.
  • If the number is a multiple of 3, it will print Fizz.
  • If the number is a multiple of 5, it will print Buzz.
  • If the number is a multiple of 7, it will print Ping.
  • If the number is a multiple of 11, it will print Pong.
  • If the number is not a multiple of any of those numbers, it will print the number.
